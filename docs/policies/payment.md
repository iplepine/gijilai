# 결제 및 가격 정책

## 가격

- 프리미엄 리포트 가격: 990원
- 결제 방식: Stripe (PaymentIntent 방식)

## 무료/유료 티어

- 무료 티어: 프리뷰/하이라이트 리포트
- 유료 티어: 전체 심층 분석 리포트
- 무료/유료 모두 동일 모델(gpt-4o-mini), 동일 프롬프트 사용. 차별화는 접근 가능한 기능으로 구분.

## 쿨다운 및 재검사

- 무료 사용자: 동일 아이 기준 7일 쿨다운
- 유료 사용자: 프리미엄 리포트 생성 후 24시간 쿨다운
- 프리미엄 리포트 구매 시 즉시 1회 재검사 기회 부여

## AI 리포트 생성

- 모델: gpt-4o-mini (전 티어 통일)
- Temperature: 0.7
- 응답 형식: JSON 객체 (response_format으로 강제)
- 3가지 프롬프트: CHILD, PARENT, HARMONY
- JSON 파싱 실패 시 원본 문자열로 폴백
- 모든 리포트 API 호출은 Supabase 세션 인증 필요
