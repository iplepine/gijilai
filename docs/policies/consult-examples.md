# 마음 통역소 고민 예시 정책

## 개요

마음 통역소 진입 시 사용자가 고민을 쉽게 입력할 수 있도록, 선택된 아이의 나이/성별에 맞는 구체적인 예시 문장을 보여준다.

## 데이터 구조

각 예시는 `{ label, text }` 형태:
- `label`: 칩 버튼에 표시되는 짧은 요약 (5~15자)
- `text`: 칩 선택 시 textarea에 채워지는 구체적 고민 문장 (20~50자)

## 예시 그룹 구조

### 나이 구간

| 구간 | 나이 | 주요 고민 영역 |
|------|------|---------------|
| 영아기 | 0~2세 | 신체 조절, 기본 애착, 수면, 식습관, 분리불안 |
| 유아기 | 3~5세 | 자아 형성, 사회성 초기, 감정 조절, 형제갈등 |
| 초등 저학년 | 6~9세 | 학교 적응, 학습 습관, 또래 관계, 규칙 이해 |
| 사춘기 | 10~13세 | 정체성, 사춘기, 디지털 미디어, 자존감 |

### 나이 판정 로직

`birthDate`로부터 만 나이(년 단위)를 계산하여 구간 결정:
- 0~2세 → `0-2`
- 3~5세 → `3-5`
- 6~9세 → `6-9`
- 10세 이상 → `10-13`

### 성별

- `gender`가 `female`이면 여아, 그 외 모든 값은 남아로 처리
- 공통 고민이 대부분이므로 성별은 보조적으로 활용
- 남아 경향: 신체적 공격성, 과잉행동, 거친 놀이, 게임 몰입
- 여아 경향: 관계 갈등, 눈치, 감정 표현, 외모 민감성

### 그룹 분류

| 그룹 | 나이 | 성별 | 예시 수 |
|------|------|------|--------|
| A1 | 0~2세 | 남아 | 10개 |
| A2 | 0~2세 | 여아 | 10개 |
| B1 | 3~5세 | 남아 | 10개 |
| B2 | 3~5세 | 여아 | 10개 |
| C1 | 6~9세 | 남아 | 10개 |
| C2 | 6~9세 | 여아 | 10개 |
| D1 | 10~13세 | 남아 | 10개 |
| D2 | 10~13세 | 여아 | 10개 |

총 예시: 80개

## 표시 규칙

- 한 번에 **5개** 표시 (칩 버튼 형태)
- 해당 아이의 나이/성별 그룹에서 **5개** 랜덤 선택 (Fisher-Yates 셔플)
- `useMemo`로 `birthDate`, `gender` 변경 시에만 재계산
- 아이 정보가 없는 경우 **3~5세 남아** 그룹에서 선택 (기본값)

## 예시 문장 작성 가이드

- 부모가 실제로 말할 법한 구어체로 작성
- 구체적인 상황이 포함되어야 함
- 감정이 담겨 있으면 좋음
- `label`: 5~15자, 핵심 키워드 중심
- `text`: 20~50자, 구체적 상황 묘사
